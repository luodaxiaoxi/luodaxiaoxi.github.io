hexo+GitHub博客搭建

发表于 2019-07-15

感谢程序羊CodeSheep的博客搭建视频。他的博客CodeSheep

这里来说一下win10下搭建博客的过程。

准备工作

1、安装git

​ 进入git官网找到Windows的下载。git安装成功的标志是在桌面上任意地方点击右键都会有Git GUI和Git Bash。

Git Bash相当于Linux的命令行操作。

2、安装node.js

​ 同样是在node.js官网下载，安装很简便一路next就行。

安装hexo博客框架

1、什么是hexo


Hexo 是一个快速、简洁且高效的博客框架。 **Hexo 使用Markdown（或其他渲染引擎）解析文章，在几秒内，即可利用靓丽的主题生成静态网页。

2、安装


在桌面任意处右键，打开Git Bash Here,得到如下界面。

由于npm速度可能比较慢，利用淘宝的镜像源安装cnpm

利用cnpm安装hexo

安装完成后可以用

3、生成本地博客

建立一个文件夹:blog

输入pwd看当前位置，当然git bash的每个命令行都会显示当前位置。这个可忽略

进入blog文件夹

初始化hexo.生成博客，这里需要大概2分钟

在桌面打开blog文件夹会发现出现了很多中间文件。

启动hexo

会出现

这就生成了一个本地网站。在浏览器打开http://localhost:4000就能出现hexo博客了。

初始化会默认有一篇名为hello world 的博客

我们可以新建一篇文章

这样就在blog/source/_posts/下生成一个名为”我的第一篇文章.md”的文章。可以用Typora等markdown编辑器编辑。

4、推向远端|将博客部署到GitHub
