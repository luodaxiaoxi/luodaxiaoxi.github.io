论文写作笔记

发表于 2019-08-05 分类于 随笔

论文写作

一、评阅原则

假设的合理性，建模的创造性；

结果的正确性，表述的清晰性。

  • 创造性：特别欣赏独树一帜，标新立异，但要合理。
  • 正确性：答案要有一定的精度，不能偏离“参考答案”很远。
  • 清晰性：摘要应理解为详细摘要，提纲挈领。

二、需要重视的问题

摘要

  • 字数控制在一页以内，半页以上；
  • 语言精炼，用词准确；
  • 阐述细致具体方法；
  • 列出主要结论；
  • 写出5-6个关键词。(方法、概念)

注：关键词用分号隔开，最后一个不加符号

问题的引出

  1. 对竞赛题中可能的模糊概念、条件作出必要的澄清
  2. 明确实际背景、已知条件、数据

问题的分析

​ 讨论什么问题，需要建立什么样的模型、用什么方法来求解做出一个定性的描述。

模型的假设

  • 严格、确切的数学语言表达
  • 假设应验证其合理性
  • 假设以3-6条为宜

例：假设题中所给数据准确无误(个别除外)

符号说明

三线表，列出全文通用符号

注：这里仅给出全文通用符号，其他符号在文中第一次出现时予以说明。

模型的建立与求解

模型的评价与检验

  1. 对结果的可靠性或误差分析
  2. 灵敏度和稳定度分析
  3. 对模型的客观评价，指出优越性和局限性
  4. 提出可能的深化，推广建议

三、多个模型的取舍问题

多个模型可选时，一至二个为宜；

实际做，可有多个，在写时选最好的，其他模型放在推广中。
